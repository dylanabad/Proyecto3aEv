# 🎯 Gestor de Colecciones

**Gestor de Colecciones** es una aplicación de escritorio desarrollada en **Java** con interfaz gráfica en **JavaFX**, pensada para coleccionistas que desean llevar un control detallado y organizado de sus objetos personales como **cómics, vinilos, videojuegos, figuras de colección**, entre otros.

---

## 🧩 Funcionalidades principales

- Registro y autenticación de usuarios.
- Creación y gestión de colecciones personales por categorías.
- Creación y gestión de items de dichas colecciones
---

## 🛠️ Tecnologías utilizadas

- **Java 17+**
- **JavaFX** para la interfaz gráfica
- **MySQL/MariaDB** como sistema gestor de base de datos
- **JDBC** para conexión con la base de datos
- Uso de DAOs, clases abstractas e interfaces para una arquitectura limpia y escalable

---

## 🎯 Objetivo del proyecto

Brindar una herramienta intuitiva y eficiente para que coleccionistas puedan gestionar sus artículos con flexibilidad, permitiendo una vista clara del valor, estado y características de sus objetos, manteniendo la información centralizada, segura y fácilmente accesible.
